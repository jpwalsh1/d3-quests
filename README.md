d3-quests
=========

Python script to check what quests in all Acts of Normal mode your level 60s have notcompleted.

# Usage #
    ./d3.py

# Sample Output #

```bash
[d3-quests]$ ./d3.py
Enter your Bnet ID, ex. Name#1234: Nicarras#1104
BeardedNic - monk - Level 60 - act1
Sword of the Stranger
```
